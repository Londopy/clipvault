A fast, keyboard-first clipboard manager daemon written in Rust.
Runs silently in the background, captures every copy, and surfaces your history in a hotkey-triggered floating overlay — no mouse required.

image::assets/demo.gif[Demo,align=center]

== Features

=== Core
* Intercepts every copy system-wide — text, images, and file paths
* Configurable ring buffer (50–1000 items) with deduplication
* Per-entry metadata: timestamp, source app, character count
* Persists history across reboots; optional session-only mode
* Single static binary — no runtime dependencies, no installer required

=== Overlay (GUI)
* Frameless floating window, always on top, appears near the cursor
* Three tabs: **History** · **Pinned** · **Snippets**
* Live fuzzy search with match highlighting
* Preview pane for long text and images
* Dismisses on focus loss or `Esc`
* Full dark / light / system / custom hex theming

=== Snippets
* Permanent named entries that survive history eviction
* Shortcode expansion — type `;;email` anywhere to expand inline
* Folder-based categories, multi-line support, JSON import/export

=== Transforms
Right-click any history entry (or press `E`) to instantly transform it:
`UPPERCASE` · `lowercase` · `Title Case` · `Sentence case` · Trim whitespace ·
Collapse newlines · URL encode/decode · Base64 encode/decode · JSON prettify/minify ·
Hex encode/decode · MD5 / SHA-1 / SHA-256 hash · Strip HTML tags · Regex find + replace

=== Security & Privacy
* App exclusion list — never record copies from specified apps (1Password, KeePassXC, …)
* Password field heuristic — auto-masks likely credentials
* Incognito mode — hotkey-toggled, no history, no disk writes
* Optional AES-256-GCM encryption of the history file
* Auto-clear history on screen lock

=== Extras
* Discord Rich Presence (live item count + session time, IPC-only — no internet)
* Auto-updater via GitHub Releases (semver check, SHA-256 verified, one-click install)
* System tray with item-count badge, pause mode, and quick-paste

== Installation

=== Binary download (recommended)

Download the latest release for your platform from
https://github.com/Londopy/clipvault/releases/latest[GitHub Releases]:

[cols="1,1,2",options="header"]
|===
|Platform |Architecture |File
|Windows  |x86-64       |`clipvault-vX.Y.Z-x86_64-windows.zip`
|macOS    |Apple Silicon|`clipvault-vX.Y.Z-aarch64-apple-darwin.dmg`
|macOS    |Intel        |`clipvault-vX.Y.Z-x86_64-apple-darwin.dmg`
|Linux    |x86-64       |`clipvault-vX.Y.Z-x86_64-unknown-linux-musl.tar.gz`
|Linux    |ARM64        |`clipvault-vX.Y.Z-aarch64-unknown-linux-musl.tar.gz`
|===

=== Package managers

[source,shell]
----
# Windows — winget
winget install clipvault

# macOS — Homebrew
brew install clipvault

# Debian / Ubuntu
sudo apt install ./clipvault_X.Y.Z_amd64.deb

# Fedora / RHEL
sudo rpm -i clipvault-X.Y.Z.x86_64.rpm
----

NOTE: Package manager entries will appear after the first stable release.

== Verifying checksums

Every release ships a `checksums.txt` signed with https://docs.sigstore.dev/[cosign].

[source,shell]
----
# Verify SHA-256
sha256sum -c checksums.txt --ignore-missing

# Verify cosign signature
cosign verify-blob --certificate clipvault.pem --signature clipvault.sig clipvault
----

== Building from source

=== Prerequisites

[cols="1,3",options="header"]
|===
|Platform |Requirements
|All      |Rust stable (`rustup toolchain install stable`)
|Windows  |Windows SDK / Visual Studio Build Tools
|macOS    |Xcode command line tools (`xcode-select --install`)
|Linux    |`libx11-dev libxi-dev libxtst-dev libdbus-1-dev pkg-config libssl-dev libxdo-dev libgtk-3-dev`
|===

=== Build

[source,shell]
----
git clone https://github.com/Londopy/clipvault.git
cd clipvault

# Debug build
cargo build

# Optimised release binary
cargo build --release

# Run directly
cargo run --release
----

== Keybind reference

=== Global (system-wide)

[cols="1,2,1",options="header"]
|===
|Keybind |Action |Platforms
|`Ctrl+Shift+V` |Open history overlay |Win · macOS · Linux
|`Ctrl+Shift+C` |Open snippets panel  |Win · macOS · Linux
|`Ctrl+Shift+X` |Clear clipboard + history |Win · macOS · Linux
|`Ctrl+Shift+P` |Paste last item (no GUI) |Win · macOS · Linux
|`Ctrl+Alt+1–9` |Instantly paste Nth item |Win · macOS · Linux
|`Ctrl+Shift+I` |Toggle incognito mode |Win · macOS · Linux
|===

NOTE: macOS uses `⌘` in place of `Ctrl` for global hooks.

=== In-overlay

[cols="1,2",options="header"]
|===
|Key |Action
|`↑` / `↓`   |Navigate history items
|`Enter`      |Paste selected item + close
|`Tab`        |Cycle: History → Pinned → Snippets
|`Del`        |Delete selected item
|`P`          |Pin / unpin selected item
|`E`          |Open transform menu
|`/`          |Jump to search bar
|`Esc`        |Close without pasting
|`1`–`9`      |Instant-paste by list index
|===

All keybinds are fully rebindable in `config.toml`.

== Configuration

Copy `config.default.toml` to the platform config directory and edit:

[cols="1,2",options="header"]
|===
|Platform |Path
|Windows  |`%APPDATA%\clipvault\config.toml`
|macOS    |`~/Library/Application Support/clipvault/config.toml`
|Linux    |`~/.config/clipvault/config.toml`
|===

[source,toml]
----
[general]
history_limit    = 200        # ring buffer depth (1–10000)
persist_history  = true
deduplicate      = true

[hotkeys]
open_history     = "ctrl+shift+v"
open_snippets    = "ctrl+shift+c"

[gui]
theme            = "dark"     # dark | light | system | custom
accent_color     = "#4f8ef7"
position         = "cursor"   # cursor | center | top-right | top-left

[security]
excluded_apps    = ["1Password", "KeePassXC", "Bitwarden"]
encrypt_history  = false      # AES-256-GCM at rest

[discord]
rich_presence    = true
application_id   = "YOUR_DISCORD_APP_ID"
----

See `config.default.toml` for the full reference with every option documented.

== Snippets

Snippets are permanent named entries accessible from the **Snippets** tab.

[source,toml]
----
# Add via config.toml or the GUI; shortcodes expand anywhere with ;;prefix
# Example shortcodes:
#   ;;email  →  your@email.com
#   ;;addr   →  123 Main Street, Springfield
#   ;;sig    →  Best regards,\nYour Name
----

== Platform notes

=== macOS

TIP: ClipVault requires Accessibility permission for global hotkeys.
On first launch you will be guided to
*System Settings → Privacy & Security → Accessibility*.

WARNING: Distributing the `.dmg` requires notarization with an Apple Developer account.
Add your credentials as GitHub Actions secrets (`APPLE_CERT`, `APPLE_CERT_PASSWORD`,
`APPLE_ID`, `APPLE_TEAM_ID`) before running the release workflow.

=== Linux (Wayland)

WARNING: Global shortcuts are not yet standardised on Wayland.
ClipVault falls back to `xdg-desktop-portal` where available; some compositors
may require additional configuration. X11 is fully supported.
